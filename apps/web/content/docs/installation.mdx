# Installation

This guide will help you install and set up PayKit in your React or Next.js project.

## Prerequisites

Before installing PayKit, make sure you have:

- Node.js 18.0 or later
- npm, yarn, or pnpm package manager
- A React or Next.js project

## Installing PayKit

### Using npm

```bash
npm install @paykit-sdk/react
```

### Using yarn

```bash
yarn add @paykit-sdk/react
```

### Using pnpm

```bash
pnpm add @paykit-sdk/react
```

## Basic Setup

### 1. Wrap your app with PayKitProvider

```tsx
import { PayKitProvider } from '@paykit-sdk/react';

function App() {
  return (
    <PayKitProvider>
      <YourApp />
    </PayKitProvider>
  );
}
```

### 2. Configure your payment providers

```tsx
import { PayKitProvider } from '@paykit-sdk/react';

function App() {
  return (
    <PayKitProvider
      config={{
        stripe: {
          publishableKey: process.env.NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY,
        },
        polar: {
          apiKey: process.env.NEXT_PUBLIC_POLAR_API_KEY,
        },
      }}
    >
      <YourApp />
    </PayKitProvider>
  );
}
```

## Environment Variables

Create a `.env.local` file in your project root and add your payment provider keys:

```env
# Stripe
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_...

# Polar
NEXT_PUBLIC_POLAR_API_KEY=pol_...

# Gumroad
NEXT_PUBLIC_GUMROAD_API_KEY=gum_...
```
